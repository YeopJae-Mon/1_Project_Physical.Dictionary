# Physical_Project
집에서도 할 수 있는 간단한 물리치료 정보전달 어플 제작
***
# 목적
```
 *sqlite를 사용하여 어플에 데이터를 반영하는 것
 *간단한 물리치료 정보를 전달하는 것
 *Play Store에 어플 출시
```
# 주요기능
```
 *물리치료 정보전달 기능
 *제작자 및 사용방법 표시
```
# 설명
```
 *본 어플은 아직 제작중에 있습니다.
 *전체적인 제작은 끝났습니다. 친구에게 디자인을 부탁하고 데이터를 채울려 합니다.
 *데이터를 다채우면 grandle를 건드려서 플레이 스토어에 출시해보려 합니다.
 *가급적이면 저작권에 안전한 사진을 사용하려 합니다.
```
